# Podcast Log

A working podcast manager built with Next.js, TypeScript, tRPC, and React Query. Started as an MVP and continuously expanding with a real database, authentication, and AI features on the roadmap.

## Why I Built This

I wanted to get comfortable with React, Next.js, TypeScript, tRPC, React Query, and Zod by actually building something real. Not a tutorial. I treated it like a take home project, started with an MVP, and I am adding advanced features to truly understand the stack.

## The Stack and Why Each Tool

- **Next.js** — React with a server already attached and a filing system for routes
- **TypeScript** — catches mistakes before the browser ever opens
- **tRPC** — one typed function call instead of endpoint, fetch, and manual types
- **React Query** — handles loading, error, cache, and refresh so you don't have to
- **Zod** — stops bad data at the door before it causes damage downstream

## What I Learned

I knew Next.js was a fullstack framework. Now I understand the server and client rendering process and how tools like tRPC and Zod connect them without looking like spaghetti code.

## Getting Started

```bash
npm run dev
```
